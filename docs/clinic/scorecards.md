# Staff scorecards

## Scorecard buckets
1. Reliability
2. Throughput
3. Safety / compliance
4. Team behavior

## Example role metrics

### Front desk
- registration accuracy
- copay capture
- schedule fill support
- no-show recovery
- phone response timeliness
- complaint rate

### Medical assistant
- room readiness
- checklist completion
- supply / stock discipline
- testing / vaccine log accuracy
- protocol adherence
- training completion

### Nurse practitioner
- note lag
- schedule utilization
- coding distribution
- refill turnaround
- prescribing exception rate
- training / policy adherence

### Office manager
- overdue task age
- action-item closure rate
- huddle completion
- audit completion
- escalation timeliness
- CAPA closure follow-through
