# Service lines

Clinic OS models these service lines separately:
- primary care / urgent-access
- women’s health
- telehealth
- weight management
- HRT
- vaccines
- waived testing
- contracted lab-in-clinic
- IV hydration
- aesthetics
- allergy testing

Each service line should eventually have:
- owner
- service charter
- inclusion/exclusion rules
- role matrix
- competency requirements
- audit tool
- pricing model
- emergency escalation sheet
- public-claims inventory
