# Clinic-specific agent organization

## Control tower
- Chief of Staff / Orchestrator
- Risk + Regulatory Gatekeeper
- Evidence Librarian / Document Control
- PMO + Dashboard Lead

## Department pods
### Office Operations
- office manager coach
- daily huddle builder
- readiness tracker
- issue triage

### People / Performance / Training
- role architect
- scorecard builder
- onboarding builder
- performance packet builder

### Quality / Accreditation
- standards mapper
- committee packet builder
- audit planner
- CAPA coordinator

### Clinical Governance (non-PHI)
- SOP / standing-order drafter
- delegation matrix builder
- service-line governance builder
- audit template builder

### Controlled Substance + Telehealth Stewardship
- prescribing governance drafter
- monitoring rules builder
- exception packet builder

### Compliance / Safety / Lab / Vaccine
- incident triage
- infection / medication safety packet builder
- contracted-service oversight builder

### Revenue / Commercial / Public Assets
- pricing packet builder
- public-copy drafter
- public-claims reviewer
- service-line economics builder

## Service-line microcells
- primary care / urgent-access
- weight management + HRT
- aesthetics
- IV hydration
- vaccines + testing

Each microcell should eventually have:
- charter
- competency matrix
- claims inventory
- emergency escalation sheet
- audit tool
- price / packaging sheet
