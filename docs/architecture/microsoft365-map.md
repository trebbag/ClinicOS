# Microsoft 365 mapping

## SharePoint
System of record for:
- policies
- SOPs
- committee packets
- evidence binders
- approved public copy archives

## Microsoft Lists
Structured tracking for:
- incidents
- CAPAs
- competencies
- action items
- issue registers
- vendor oversight
- manual import status

## Planner
Task distribution for:
- office manager routines
- committee follow-ups
- remediation tasks
- rollout tasks

## Teams
Notification surface for:
- approval nudges
- daily huddles
- committee meetings
- status alerts

## Power Automate / approvals
Human-in-the-loop decision routing for:
- document approvals
- public copy approvals
- action approvals
